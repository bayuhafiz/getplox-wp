=== WooCommerce Shipwire ===
Author: woothemes, skyverge
Tags: woocommerce
Requires at least: 4.1
Tested up to: 4.5.2
Requires WooCommerce at least: 2.4.13
Tested WooCommerce up to: 2.6.0

A full-featured Shipwire integration for WooCommerce, including real-time shipping rates, automatic order fulfillment processing, and live inventory / tracking updates.

See http://docs.woothemes.com/document/shipwire/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-shipwire' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
